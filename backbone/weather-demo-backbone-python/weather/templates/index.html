Weather App Template

Modify this file at: weather/templates/index.html

Get weather forecasts with: /forecast/?location=<city name or zip>

The Cities REST service is at: /cities/

To view the REST service data use the Admin Page. (user: admin , password: admin)

jQuery, Underscore, and Backbone are already included in this template, which is the bare minimum you should need to get started.
